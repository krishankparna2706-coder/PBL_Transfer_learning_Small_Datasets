Transfer Learning for Small Datasets

"To demonstrate that transfer learning can achieve high accuracy and generalization on small image datasets (e.g., ants vs bees) by using a pre-trained CNN as a feature extractor with a trainable classification head, reducing data and compute requirements."

Problem Statement

Training deep neural networks from scratch requires large labeled datasets and significant compute. In many real-world scenarios—medical imaging, niche industrial inspection, or small-team projects—only a small amount of labeled data is available. This project addresses the challenge of achieving reliable image classification with limited data by leveraging transfer learning.

Literature Review / Market Research

Existing approaches include: training small custom CNNs (poor generalization on small data), using pre-trained models with full fine-tuning (risk of overfitting), and feature extraction from pre-trained backbones (e.g., ResNet, VGG on ImageNet). PyTorch and TensorFlow provide standard pipelines for transfer learning; the PyTorch transfer learning tutorial and similar guides establish baseline practices (freeze backbone, train head, optional fine-tuning).

Research Gap / Innovation

Our approach focuses on a minimal, reproducible MVP: frozen ResNet18 backbone + data augmentation (random crop, flip, normalization) + a single trainable classification head. We package this into a runnable project with training, CLI inference, and a REST API, so anyone can replicate and extend the pipeline for their own small-dataset classification task.

System Methodology

Dataset / Input

Source: Hymenoptera dataset (ants vs bees) from PyTorch tutorials. Structure: train/ and val/ with class subfolders.

Size: Small dataset (~120 training images per class); suitable for demonstrating transfer learning.

Preprocessing: Resize to 256, RandomResizedCrop(224) and RandomHorizontalFlip for training; CenterCrop(224) for validation. ImageNet normalization (mean [0.485, 0.456, 0.406], std [0.229, 0.224, 0.225]).

Model / Architecture

Backbone: ResNet18 pre-trained on ImageNet (frozen).

Head: Single linear layer mapping backbone features (512) to number of classes (2 for ants/bees).

Training: CrossEntropyLoss, SGD (momentum 0.9), StepLR (step 7, gamma 0.1). Only the new classification head is trained.

Live Execution

VIEW CODE / DEMO → GitHub Repository

Clone the repo, install dependencies, run python main.py to train, and python app.py to start the inference API.

Results & Analysis

Accuracy / Performance: ≥ 85% validation accuracy on hymenoptera_data (MVP success criterion). Exact metrics (accuracy, loss, classification report, confusion matrix) are printed at the end of training.

Quantifiable outcomes: transfer learning with a frozen backbone and minimal augmentation consistently reaches the target accuracy; the framework is adaptable to other small-dataset classification tasks by changing the dataset path and number of classes.
